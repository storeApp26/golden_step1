🔥 إعداد Firebase

هذا الإعداد مرة واحدة فقط — بعده المتجر يعمل على الإنترنت لجميع الزبائن.

1

أنشئ مشروع Firebase مجاني

اذهب إلى console.firebase.google.com → أنشئ مشروعاً جديداً

2

فعّل Firestore Database

من القائمة → Firestore Database → Create Database → Test mode → Enable

3

احصل على إعدادات المشروع

⚙️ Project Settings → General → Your apps → </> → انسخ الـ Config
